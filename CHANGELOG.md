# Changelog

所有版本变更记录请参见 [Releases](https://github.com/nevertiree/zhihusync/releases)。

## [1.0.0] - 2026-03-07

### 🎉 正式发布

- 项目首次正式发布 v1.0.0 版本
- 引入标准化版本管理机制
- 引入标准化分支管理流程 (Git Flow)

### ✨ 主要功能

- 知乎点赞内容自动备份
- Docker 一键安装部署
- Web 管理界面
- 多平台支持 (Linux/Windows/macOS)

### 🔧 技术栈

- Python 3.10+
- FastAPI + SQLAlchemy
- Playwright 浏览器自动化
- Docker + Docker Compose
