🖥️ 部署环境

💡 NAS Docker 部署存储映射：
  • HTML存储: ./data/html → /app/data/html
  • 数据库: ./data/meta → /app/data/meta
  • 静态文件: ./data/static → /app/data/static
  • 配置文件: ./config → /app/config
💡 Windows 本地部署：
  • 数据目录: ./data (程序同级目录)
  • 配置文件: ./config/config.yaml
  • 建议使用 PowerShell 或 CMD 运行
  • 需要安装 Playwright: playwright install chromium
💡 Linux 本地部署：
  • 数据目录: ~/.zhihusync/data 或 ./data
  • 配置文件: ~/.zhihusync/config.yaml 或 ./config/config.yaml
  • 建议使用 systemd 服务运行
  • 需要安装浏览器依赖: playwright install-deps chromium

🔑 第一步：配置 Cookie（必需）

知乎需要登录后才能访问点赞内容，请按以下步骤获取 Cookie：

  1. 打开 Chrome 浏览器，访问 zhihu.com 并登录
  2. 按 F12 或 Ctrl+Shift+I 打开开发者工具
  3. 切换到 Application/应用 标签
  4. 在左侧选择 Cookies → https://www.zhihu.com
  5. 右键点击表格 → Copy all / 复制全部
  6. 将内容粘贴到下方的 Cookie 输入框中
  1. 打开 Edge 浏览器，访问 zhihu.com 并登录
  2. 按 F12 或 Ctrl+Shift+I 打开开发者工具
  3. 切换到 应用程序 标签
  4. 在左侧选择 Cookie → https://www.zhihu.com
  5. 右键点击表格 → 复制全部
  6. 将内容粘贴到下方的 Cookie 输入框中
  1. 打开 Firefox 浏览器，访问 zhihu.com 并登录
  2. 按 F12 或 Ctrl+Shift+K 打开开发者工具
  3. 切换到 存储 标签
  4. 在左侧选择 Cookies → https://www.zhihu.com
  5. 右键点击表格 → 复制全部
  6. 将内容粘贴到下方的 Cookie 输入框中
  1. 打开 Safari，偏好设置 → 高级 → 勾选"在菜单栏中显示开发菜单"
  2. 访问 zhihu.com 并登录
  3. 点击菜单栏 开发 → 显示 JavaScript 控制台
  4. 切换到 存储 标签
  5. 选择 Cookies → zhihu.com
  6. 导出或手动复制 Cookie 值
💡 替代方案：安装 EditThisCookie 浏览器扩展，点击扩展图标 → Export → 复制 JSON 格式

👤 第二步：设置用户 ID（必需）

用户 ID 用于识别要备份哪个账号的点赞内容：

  1. 访问你的知乎个人主页
  2. 从 URL 中提取用户 ID
    例如：https://www.zhihu.com/people/zhang-san-123
    用户 ID 就是 zhang-san-123
  3. 切换到"基本设置"标签页，将用户 ID 填入输入框

❓ 常见问题

Cookie 有效期多久？

知乎 Cookie 通常有效期为 1-3 个月，过期后需要重新配置。如果同步失败且日志显示登录失效，请重新获取 Cookie。

如何找到自己的用户 ID？

登录知乎后，点击右上角头像 → 我的主页，地址栏中的 URL 格式为 https://www.zhihu.com/people/xxx，其中的 xxx 就是你的用户 ID。

同步失败怎么办？

1. 检查 Cookie 是否有效
2. 检查用户 ID 是否正确
3. 查看日志页面的错误信息
4. 尝试增加请求间隔时间

备份的数据在哪里？

备份的 HTML 文件保存在 data/html 目录下，数据库文件在 data/meta/zhihusync.db。
