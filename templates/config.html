📖 配置指南

🔑 第一步：配置 Cookie（必需）

知乎需要登录后才能访问点赞内容，请按以下步骤获取 Cookie：

  1. 在电脑浏览器中打开 zhihu.com 并登录
  2. 按 F12 打开开发者工具
  3. 切换到 Application/应用 标签
  4. 在左侧选择 Cookies → https://www.zhihu.com
  5. 复制所有 Cookie 内容（或使用 EditThisCookie 插件导出 JSON）
  6. 将内容粘贴到下方的 Cookie 输入框中
💡 提示：Cookie 会定期过期，如果同步失败，请重新获取并更新。

👤 第二步：设置用户 ID（必需）

用户 ID 用于识别要备份哪个账号的点赞内容：

  1. 访问你的知乎个人主页
  2. 从 URL 中提取用户 ID
    例如：https://www.zhihu.com/people/zhang-san-123
    用户 ID 就是 zhang-san-123
  3. 将用户 ID 填入下方输入框

⚙️ 第三步：高级设置（可选）

  • 扫描间隔：自动同步的时间间隔（分钟），建议 30-120 分钟
  • 最大条目数：每次同步最多获取多少条新内容
  • 保存评论：是否同时备份评论内容

📋 如何获取 Cookie？（点击查看详细步骤）
方法一：手动复制（推荐）
  1. 在浏览器中登录知乎
  2. 按 F12 打开开发者工具
  3. 切换到 Application/应用 标签
  4. 选择 Cookies → https://www.zhihu.com
  5. 右键点击表格 → Copy all / 复制全部
方法二：使用 EditThisCookie 插件
  1. 安装 EditThisCookie 浏览器扩展
  2. 登录知乎后，点击扩展图标
  3. 选择 "Export" → "Export as JSON"
  4. 复制导出的 JSON 内容
方法三：使用控制台命令
// 在 Console 标签页执行
JSON.stringify(document.cookie.split(';').map(c => {
  const [n, ...v] = c.trim().split('=');
  return {name: n, value: v.join('='), domain: '.zhihu.com'};
}))

❓ 常见问题

Cookie 有效期多久？

知乎 Cookie 通常有效期为 1-3 个月，过期后需要重新配置。如果同步失败且日志显示登录失效，请重新获取 Cookie。

如何找到自己的用户 ID？

登录知乎后，点击右上角头像 → 我的主页，地址栏中的 URL 格式为 https://www.zhihu.com/people/xxx，其中的 xxx 就是你的用户 ID。

同步失败怎么办？

1. 检查 Cookie 是否有效
2. 检查用户 ID 是否正确
3. 查看日志页面的错误信息
4. 尝试增加请求间隔时间

备份的数据在哪里？

备份的 HTML 文件保存在 data/html 目录下，数据库文件在 data/meta/zhihusync.db。
